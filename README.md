# Research Talent & Capability OS

A static prototype for an AI-powered Research Talent & Capability Operating System.

## Run locally

Open `index.html` directly, or serve the folder:

```bash
python3 -m http.server 58917
```

Then visit:

```text
http://127.0.0.1:58917/index.html
```

## Deploy on GitHub Pages

Push `main` to GitHub, then open repository **Settings → Pages** and choose:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/ (root)**

GitHub Pages will publish the static site from `index.html`.
